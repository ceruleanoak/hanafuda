# Audio Files Directory

This directory contains music and sound effects for the Hanafuda game.

## Required Audio Files

Place the following audio files in this directory:

### Win/Lose Music

- **hanafuda-win.mp3** - Music played when the player wins the game
  - Recommended duration: 2-3 seconds
  - Should have a celebratory, triumphant feel
  - Will play during the winning animation

- **hanafuda-lose.mp3** - Music played when the player loses the game
  - Recommended duration: 1.5-2 seconds
  - Should have a disappointed or sympathetic feel
  - Will play during the losing animation

## Audio Format

- **Format:** MP3 (recommended for best browser compatibility)
- **Alternative formats:** OGG, WAV also supported
- **Sample rate:** 44.1 kHz or 48 kHz recommended
- **Bitrate:** 128-192 kbps recommended (good quality/size balance)

## Notes

- If audio files are missing, the game will continue to work but without sound
- Audio can be enabled/disabled in the game options menu
- Volume can be adjusted (default is 50%)
